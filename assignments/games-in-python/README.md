
# 📘 Assignment: Hangman Game

## 🎯 Objective

Students will build a text-based Hangman game in Python that reinforces string operations, loops, conditionals, and simple game logic.

## 📝 Tasks

### 🛠️ Task 1: Build Hangman core mechanics

#### Description
Create a Python program that selects a random secret word and allows the player to guess one letter at a time.

#### Requirements
Completed program should:

- Randomly select a secret word from a predefined list
- Display the current word progress with blanks for unguessed letters (e.g. `_ a _ g _ a n`)
- Accept single-letter input and validate guesses
- Track and show incorrect guesses and remaining attempts
- End when the word is fully guessed or attempts are exhausted

### 🛠️ Task 2: Add win/lose flow and replay option

#### Description
Extend the game to show final messages and let players choose to play again.

#### Requirements
Completed program should:

- Print a win message with the full word if the player succeeds
- Print a lose message and reveal the word if attempts run out
- Prompt the player to replay or quit after each game
